***python multiline-comment-workaround:***  
edit your vscode settings.json:  

"editor.tokenColorCustomizations": {  
        "[Ayu Mirage PowerUp]":{  
          "textMateRules":[  
            {  
              "scope": "string.quoted.docstring.multi.python",  
              "settings":{  
                "foreground": "#5c6773"  
               }  
            }  
          ]  
        }  
    }  
}
